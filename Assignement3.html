Html for Lists

1. Start tags for

  • an ordered list
    • "ol" "li"Coffee"/li" "li"Tea"/li" "li"Milk"/li" "/ol"
  • an unordered list
    • "ul" "li"Coffee"/li" "li"Tea"/li" "li"Milk"/li" "/ul"
  • a description list
    • "dl" "dt"Coffee"/dt" "dd"Tea"/dd" "dd"Milk"/dd" "/dl"
  • a list item

    • Each list item starts with the < li > tag.

  • a description term

    • the < dt > tag defines the term (name).

  • a description data

    • the < dd > tag describes each term.

  • a table header

    • Each table header is defined with the < thead > tag.

  • a table header cell

    • A table header is defined with the < th > tag.

  • a table body

    • A table body is defined with the < tbody > tag.

  • a table cell

    • A table cell is defined with the < td > tag.

  • a table footer

    • A table footer is defined with the < tfoot > tag.
